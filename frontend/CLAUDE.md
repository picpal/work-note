# work-note — 프로젝트 메모

폐쇄망 사내용 마크다운 웹에디터. 초기에는 개인 PC + SQLite(프로토타입에서는 localStorage로 대체), 이후 사내 서버 공용 서비스로 확장 예정.

## 프론트엔드 아키텍처 원칙 (사용자 요청)
- **컴포넌트화**를 철저히: 관심사별로 파일 분리, 작게 유지.
- 이 프로젝트에 맞는 **디자인 패턴**을 적용:
  - 단일 진실 공급원 + **Reducer 패턴**으로 vault(트리) 상태 일원화, 불변 업데이트.
  - **커스텀 훅**으로 로직 분리: `useVault`, `usePersist`, `useContextMenu`, `useTheme`.
  - **커맨드 패턴**으로 내보내기/툴바 액션 분리.
  - 표현(컴포넌트) / 상태·부수효과(훅) 분리.

## 디자인
- 순수 모노톤(따뜻한 뉴트럴 그레이). 라이트 기본 + 다크. 액센트 없음.
- Pretendard(UI/본문) + D2Coding(코드). 가독성 최우선.
- 에디터: Obsidian식 라이브 프리뷰. 사이드바: 아코디언 트리 + 우클릭 + Cmd+K 검색.
- mermaid / 시퀀스 다이어그램 / 코드 하이라이팅 / 내보내기(PDF·MD·클립보드) 지원.

> 프로토타입은 CDN 라이브러리를 사용. **실제 폐쇄망 배포 시 marked·highlight.js·mermaid·폰트를 로컬 번들로 교체**해야 함.
